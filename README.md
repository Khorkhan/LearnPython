### รายละเอียดโปรเจกต์

Repository นี้เป็นที่รวบรวมแบบฝึกหัดและโปรเจกต์ขนาดเล็กที่ผมสร้างขึ้นในระหว่างการเรียนรู้ภาษา Python โดยอ้างอิงเนื้อหาจากหลักสูตร **Learn the Basics** ของ [learnpython.org](https://www.learnpython.org/)

### สิ่งที่ได้เรียนรู้

ผมได้ศึกษาและลงมือทำในหัวข้อพื้นฐานที่สำคัญ ดังนี้:

* **Variables and Types**: การประกาศตัวแปรและชนิดข้อมูลพื้นฐาน
* **Lists**: การจัดการรายการข้อมูล
* **Basic Operators**: การคำนวณและการดำเนินการทางคณิตศาสตร์
* **String Formatting**: การจัดรูปแบบข้อความให้สวยงาม
* **Basic String Operations**: การจัดการและตัดต่อข้อความ
* **Conditions**: การสร้างเงื่อนไข (If/Else)
* **Loops**: การวนซ้ำ (For/While)
* **Functions**: การสร้างฟังก์ชันเพื่อใช้งานซ้ำ
* **Classes and Objects**: พื้นฐานการเขียนโปรแกรมเชิงวัตถุ (OOP)
* **Dictionaries**: การเก็บข้อมูลแบบ Key-Value
* **Modules and Packages**: การนำโค้ดจากไฟล์อื่นมาใช้งาน

## 🇺🇸 English Version

### Project Description

This repository contains exercises and small projects I developed while learning Python, based on the **Learn the Basics** curriculum from [learnpython.org](https://www.learnpython.org/).

### Skills Acquired

I have explored and implemented the following fundamental concepts:

* **Variables and Types**: Variable declaration and data types.
* **Lists**: Managing collections of data.
* **Basic Operators**: Arithmetic operations.
* **String Formatting**: Professional string output.
* **Basic String Operations**: String manipulation and slicing.
* **Conditions**: Logic and decision-making (If/Else).
* **Loops**: Iteration using For and While loops.
* **Functions**: Creating reusable code blocks.
* **Classes and Objects**: Object-Oriented Programming (OOP) basics.
* **Dictionaries**: Working with Key-Value pairs.
* **Modules and Packages**: Organizing code and importing external libraries.

### Sample Projects

This repository includes practical implementations:

1. **Quick Note**: A simple note-taking app with File I/O.
2. **Currency Converter**: A tool for converting currencies using Dictionaries.
3. **Guessing Game**: An interactive game utilizing the `random` module and loops.
